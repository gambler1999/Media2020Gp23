Calculation of Random Event


Deﬁnition:

To record the experiment data, we use these two main paramaters:


  • Each possible outcome: sample point (ω)

  • The set of all possible outcomes: sample space (Ω)

Any subset of the sample space Ω is known as an random event or event. Events are usually denoted by capital letters A,B,C,···. Those events must occur in the experiment are called the inevitable events. Sample space Ω is an inevitable event. Those not happen anytime are said to be impossible events. We usually denote impossible events by ∅.

Basic rules:

Let A,B and C be the random events of experiment E. The operations of the events will satisfy the following rules:


Commutatively: ${A}\bigcup{B}$ = ${B}\bigcup{A}$, $AB = BA$
Associatively: ${A}\bigcup{({B}\bigcup{C})}$= ${({A}\bigcup{B})}\bigcup{C}$ =${A}\bigcup{B}\bigcup{C}$. $A(BC) = (AB)C = ABC$.

Distributively: ${A}\bigcup{({B}\bigcap{C})} = {({A}\bigcup{B})}\bigcap{({A}\bigcup{C})}$, ${A}{({B}\bigcup{C})} = {({A}\bigcap{B})}\bigcup{({A}\bigcap{C})}$

De Morgan’s law: ${A}\bigcup{\overline{B}}$ = ${\overline{A}}\bigcap{\overline{B}}$, ${A}\bigcap{\overline{B}}$ = ${\overline{A}}\bigcup{\overline{B}}$

In addition, there are some common properties, such as


$\overline{\overline{A}}$ = ${A}$

${{A}\bigcap{B}}\subset{A}$ and ${{A}\bigcap{B}}\supset{B}$

${{A}\bigcup{B}}\supset{A}$ and ${{A}\bigcup{B}}\subset{B}$

$A−B$ = $A−AB$ = $\overline{AB}$

${A}\bigcup{B}$ = ${A}\bigcup{{\overline{A}}B}$

Example:

Suppose that A,B and C are three events and D = {at least one of the three events will occur}. Try to describe the event D by events A,B and C.}


Solution:


We describe the event D by three diﬀerent ways:

  1. directed method: $D = ABC$,

  2. decomposition method: $D = {(AB)C}\bigcap{ABC}\bigcap{A(BC)}$

  3. inverse method: $D = (A,B and C will not occur) = ABC$.
